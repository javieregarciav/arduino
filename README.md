# MatrizLED

Web app para enviar mensajes a una matriz de LEDs 8×8 conectada a un Arduino UNO R4. El Arduino consulta una API en AWS, y este sitio publica los mensajes que se ven en la matriz en tiempo real.

**🔗 [arduino-six.vercel.app](https://arduino-six.vercel.app/)**

## Stack

- React + TypeScript + Vite
- Tailwind CSS v4
- API REST en AWS API Gateway + Lambda

## Desarrollo

```bash
npm install
npm run dev
```
